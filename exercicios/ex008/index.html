Principais formatações

Negrito ou destaque

Nesta frase temos um termo em negrito usando a tag B, (que é NÃO semântica).

Nesta frase tenho um termo em destaque usando a tag STRONG (Semântica).

Italíco / Ênfase

Nesta frase temos um termo em itálico, usanto a tag I (que é NÃO semântica)

Nesta frase termo um termo em destaque italíco , usando a tag EM (Semântica)

Texto marcado

Podemos criar também um texto marcado usando a tag MARK do HMTL.

Usando a tag do texto marcado na cor original amarelo. Style é usado em CSS

Podemos também criar uma tag style, no cabeçalho, abaixo do título, crio um seletor para MARK, ver linha 7 padronizando assim todas as tags MARK no meu HTML.

Usando outras cores para marcar um texto

Lembrando que a tag style é utilizada em CSS, mas conseguimos utilizar a tag MARK.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo.

A tag BIG, está obsoleta (pois é forma), ficou em vermelho na minha HTML, já a tag SMALL não está obsoleta (podemos nos referir as letras pequenas de um contrato, conseguimos dar sentido de usar a letra pequena, é semântica).

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois.

Existe também o sublinhado usando a tag U (NÃO semântica).

Texto sobrescrito

Usado para iserir algo do tipo: x20+3. Usamos a tag SUP.

Texto subscrito

Usado para inserir algo do tipo H2O. Usamos a tag SUB.
